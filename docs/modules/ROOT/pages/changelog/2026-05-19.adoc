== CloudNativePG is the new default

All new PostgreSQL instances will use CloudNativePG. 
Enjoy your new https://docs.appcat.ch/vshn-managed/postgresql/create.html[PostgreSQL].
If you have not done it yet, now is the time to migrate your existing PostgreSQL instances to CloudNativePG. You can do it on your own by following this https://docs.appcat.ch/vshn-managed/postgresql/migration.html[migration guide]. Reach out to us if you have questions.
